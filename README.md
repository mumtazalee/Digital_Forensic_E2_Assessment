# Digital_Forensic_E2_Assessment
453779 Element 2 Napoleon Bonaparte
Scenario/task
You are working for a national security services agency. You have been tasked with part of the vetting process for a potential new employee, Mr Napoleon Bonaparte
A forensic image has been taken from his laptop. Your task is to analyse it in order to build a profile of the person and to ascertain whether he has been involved in any criminal activities or anything that could potentially compromise his position if he was appointed to this sensitive role.
